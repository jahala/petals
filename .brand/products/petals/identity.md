## Mark (evolved 2026-06-12)

Three slim petals fanned from one base point — vital leaf `#3F7A33`, vivid blossom `#C04B3C`, luminous petal `#E8917F` — beside the lowercase wordmark "petals" set whole in ink. The mark carries the palette itself; the word stays calm. It echoes the three skills (extract / distribute / check) and says the name: petals, plural. Replaces the single-petal mark, which replaced the four-petal pinwheel.

## Logo Usage

- **Canonical asset**: `.brand/products/petals/assets/logo.svg`. Copy it byte-for-byte — never redraw, trace, or approximate the paths.
- **Fills are exact**: `#3F7A33` / `#C04B3C` / `#E8917F`, in that drawing order. Recoloring the mark is off-brand. One exception: an all-ink (`#20241B`) mono variant where color is unavailable (favicons on unknown surfaces, single-color print).
- **Minimum size**: 20px mark height in digital use. Below that, use the mono variant or the wordmark alone.
- **Clear space**: at least half the mark's height on all sides.
- **Never**: stretch, rotate, outline, add shadows or effects, place on backgrounds that fail the contrast pairings in `colors.md`, or use a single petal of the fan as the logo.
- **Wordmark**: lowercase "petals", set whole in the heading serif, ink. Never split, never letter-spaced apart, never title-cased.
